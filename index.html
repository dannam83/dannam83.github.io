Inspired by Yelp

Eatery is a full stack web application built with Ruby on Rails in the backend and React-Redux on the frontend. It features password encryption for accounts, Google Maps API implementation, and forms allowing users to post their own restaurant reviews.

Wall-E Ball

Inspired by Doodle Jump

Wall-E Ball is a purely JavaScript web app. It utilizes JavaScript Canvas and requestAnimationFrame to render a seamless scrolling 2D motion experience. Euclidean vectors are used to modify Wall-E's movements and to simulate gravity.

A tutorial exercise on url requests and responses

At Your Service is an exercise that goes through examples of server and controller functions. By being equipped with a customized server and controller for each exercise, the user can observe how requests and responses are handled with differing url inputs.
